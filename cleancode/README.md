# 클린 코드
이 책은 깨끗한 변수 이름, 깨끗한 함수, 깨끗한 클래스를 만드는 방법을 소개한다.
사례 코드를 더 좋은 코드로 개선하는 과정을 통해 저자의 의도와 사고방식을 이해하도록 해보자.
